### Nuxt Image Package
This package allows you to use img tag with src and alt props. Just it!
<br>**https://www.npmjs.com/package/n-img-v**

<hr>

#### # Installation
```
yarn add n-img-v
```

#### # Usage
In script
```
import NImgV from 'n-img-v'
```

Inside template
```
<n-img-v
  :src="{image_url}"
  :alt="{string}"
/>
```
